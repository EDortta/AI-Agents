# 08 - Flujo Intermedio de Equipo

## Historia feliz: equipo sincronizado
Lia organiza roles claros y usa el agente como acelerador, no como dueño de decisiones.

## Roles
- Planner: define alcance y prioridad
- Programmer: implementa
- Reviewer: valida riesgo/regresión
- Agente: acelera ejecución y documentación

## Ciclo de etapa
1. Definir work_id y alcance.
2. Planear con agente.
3. Implementar en incrementos cortos.
4. Revisar evidencia.
5. Cerrar sesión con handoff + napkin.

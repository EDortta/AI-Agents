# 02 - Primera Tarea con Agente de IA

## Historia feliz: alcance corto, entrega limpia
Lia toma una tarea pequeña. Evita prompts vagos. Pide plan primero y luego implementación con evidencia.

## Qué es tuyo (programador)
- Definir alcance corto.
- Aprobar plan antes de editar.
- Exigir evidencia de tests/checks.

## Qué es del agente
- Traducir pedido a plan técnico.
- Implementar cambio mínimo y seguro.
- Ejecutar y reportar validaciones.

## Flujo práctico
1. Objetivo + contexto + restricciones + criterio de terminado.
2. Pedir plan.
3. Aprobar plan.
4. Pedir implementación.
5. Pedir archivos cambiados + tests + riesgos.

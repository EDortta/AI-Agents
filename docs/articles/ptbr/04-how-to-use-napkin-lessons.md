# 04 - Como Usar napkin-lessons.md

## História feliz: erro virou melhoria
Lia percebeu que perdeu tempo por não validar um endpoint antes de commitar. Ela registrou isso em `docs/napkin-lessons.md`. Na próxima etapa, o erro não se repetiu.

## O que é seu (programador)
- Capturar lições curtas e úteis.
- Sempre escrever "ação para próxima vez".

## O que é do agente
- Ajudar a transformar eventos em lições objetivas.
- Sugerir ações de prevenção.

## Fórmula simples
- Lição: o que aconteceu.
- Ação: o que você fará diferente.

## Exemplo bom
- "[2026-04-20] WK-20260420-auth - Esqueci teste de erro 401."
- "Action next time: incluir cenário negativo no Test Plan antes de implementar."

## Exemplo ruim
- "Preciso melhorar."

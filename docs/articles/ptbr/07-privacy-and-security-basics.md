# 07 - Fundamentos de Privacidade e Segurança

## História feliz: feature segura desde o início
Lia implementa um cadastro com dados pessoais. Em vez de corrigir depois, ela já pede avaliação de privacidade e segurança na mesma tarefa.

## O que é seu (programador)
- Sinalizar quando há dados pessoais.
- Exigir checklist de privacidade e segurança.

## O que é do agente
- Aplicar `docs/agents/privacy-compliance.md`.
- Aplicar `docs/agents/security.md`.
- Reportar risco, mitigação e risco residual.

## Verificações mínimas
- minimização de dados
- retenção/remoção
- sem log sensível
- controle de acesso

## Prompt sugerido
"Esta tarefa envolve dados pessoais. Execute revisão de privacidade e segurança e reporte riscos com mitigação."

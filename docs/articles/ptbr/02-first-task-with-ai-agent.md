# 02 - Primeira Tarefa com Agente de IA

## História feliz: primeira entrega limpa
Depois do setup, Lia escolhe uma tarefa pequena. Ela não pede "faz tudo". Ela pede uma mudança objetiva, com teste e evidência. Resultado: entrega rápida e sem confusão.

## O que é seu (programador)
- Definir escopo curto (uma tarefa por vez).
- Aprovar o plano antes de editar arquivos.
- Exigir evidência de teste/check.

## O que é do agente
- Traduzir pedido em plano técnico.
- Implementar mudança com mínimo impacto.
- Executar e reportar validações.

## Passo a passo
1. Escreva objetivo + contexto + restrições + critério de pronto.
2. Peça plano primeiro.
3. Aprove o plano.
4. Peça implementação.
5. Peça resumo com arquivos alterados, testes e riscos.

## Prompt sugerido
"Objetivo X. Contexto Y. Restrições Z. Proponha plano em 5 passos, espere aprovação, implemente e mostre testes executados."

## Erro comum
- Iniciante pular direto para implementação sem plano.

## Sinal de sucesso
- Mudança pequena, validada e reversível.

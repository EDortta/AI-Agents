# 06 - Work ID and Commit Traceability

## Happy story: audit in minutes
One week later, Lia needs to explain a change. Because work_id is linked everywhere, traceability is immediate.

## Format
`WK-YYYYMMDD-<short-slug>`

## Where to use
- epic/task docs
- handoff entries
- commit messages

## Value
Planning, implementation, and history stay connected.

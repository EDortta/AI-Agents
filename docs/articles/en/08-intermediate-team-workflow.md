# 08 - Intermediate Team Workflow

## Happy story: team alignment
Lia's team split roles clearly and used the agent as accelerator, not decision owner.

## Roles
- Planner: scope and priority
- Programmer: implementation
- Reviewer: risk/regression validation
- Agent: execution support and documentation

## Stage cycle
1. Define work_id and scope.
2. Plan with agent.
3. Implement in small increments.
4. Review evidence.
5. Session-close update.

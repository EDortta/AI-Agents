# 07 - Privacy and Security Basics

## Happy story: safe by design
Lia implements a feature with personal data. She asks for privacy + security review before merging.

## What is yours (programmer)
- Explicitly mark personal-data impact.
- Require privacy and security checks.

## What is the agent's
- Apply `docs/agents/privacy-compliance.md`.
- Apply `docs/agents/security.md`.
- Report risk, mitigation, residual risk.

## Minimum checks
- data minimization
- retention/deletion
- no sensitive logs
- access boundaries

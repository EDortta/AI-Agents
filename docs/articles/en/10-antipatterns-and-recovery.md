# 10 - Anti-patterns and Recovery

## Happy story: from chaos to control
Lia inherited messy work: no work_id, no handoff, weak validation. She recovered using a simple protocol.

## Anti-patterns
- coding before limits are ready
- vague huge prompts
- ending stage without handoff
- generic lessons with no action

## Recovery protocol
1. Rebuild status in `handoff.md`.
2. Normalize work_id mapping.
3. Re-check limits and scope.
4. Run minimum checks and document.
5. Add actionable napkin lessons.

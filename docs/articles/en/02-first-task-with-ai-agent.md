# 02 - First Task with an AI Agent

## Happy story: small scope, clean delivery
Lia picks one small task. She avoids "do everything" prompts. She asks for plan first, then implementation with test evidence.

## What is yours (programmer)
- Define short scope.
- Approve plan before edits.
- Require test/check evidence.

## What is the agent's
- Convert request into technical plan.
- Implement minimal safe change.
- Run and report validations.

## Practical flow
1. Goal + context + constraints + done criteria.
2. Ask for plan.
3. Approve plan.
4. Request implementation.
5. Request changed files + tests + risks.
